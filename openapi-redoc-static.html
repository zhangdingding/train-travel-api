Stations

Find and filter train stations across Europe, including their location and local timezone.

Get a list of train stations

Returns a paginated and searchable list of all train stations.

Authorizations:
OAuth2
query Parameters
page
integer >= 1
Default: 1
Example: page=1

The page number to return

limit
integer [ 1 .. 100 ]
Default: 10
Example: limit=10

The number of items to return per page

coordinates
string
Example: coordinates=52.5200,13.4050

The latitude and longitude of the user's location, to narrow down the search results to sites within a proximity of this location.

search
string
Examples: search=Milano Centrale search=Paris

A search term to filter the list of stations by name or address.

country
string <iso-country-code>
Example: country=DE

Filter stations by country code

Trips

Timetables and routes for train trips between stations, including pricing and availability.

Get available train trips

Returns a list of available train trips between the specified origin and destination stations on the given date, and allows for filtering by bicycle and dog allowances.

Authorizations:
OAuth2
query Parameters
page
integer >= 1
Default: 1
Example: page=1

The page number to return

limit
integer [ 1 .. 100 ]
Default: 10
Example: limit=10

The number of items to return per page

origin
required
string <uuid>
Example: origin=efdbb9d1-02c2-4bc3-afb7-6788d8782b1e

The ID of the origin station

destination
required
string <uuid>
Example: destination=b2e783e1-c824-4d63-b37a-d8d698862f1d

The ID of the destination station

date
required
string <date-time>
Example: date=2024-02-01T09:00:00Z

The date and time of the trip in ISO 8601 format in origin station's timezone.

bicycles
boolean
Default: false

Only return trips where bicycles are known to be allowed

dogs
boolean
Default: false

Only return trips where dogs are known to be allowed

Payments

Pay for bookings using a card or bank account, and view payment status and history.

warn Bookings usually expire within 1 hour so you'll need to make your payment before the expiry date

Pay for a Booking

A payment is an attempt to pay for the booking, which will confirm the booking for the user and enable them to get their tickets.

Authorizations:
OAuth2
path Parameters
bookingId
required
string <uuid>
Example: 1725ff48-ab45-4bb5-9d02-88745177dedb

The ID of the booking to pay for.

Request Body schema: application/json
required

Payment details

amount
number > 0

Amount intended to be collected by this payment. A positive decimal figure describing the amount to be collected.

currency
string
Enum: "bam" "bgn" "chf" "eur" "gbp" "nok" "sek" "try"

Three-letter ISO currency code, in lowercase.

Card (object) or Bank Account (object)

The payment source to take the payment from. This can be a card or a bank account. Some of these properties will be hidden on read to protect PII leaking.
